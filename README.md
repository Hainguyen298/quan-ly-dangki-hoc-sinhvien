# Hệ Thống Quản Lý Đăng Ký Học Của Sinh Viên

## Mô tả
Chương trình quản lý đăng ký học phần của sinh viên được xây dựng bằng C++.

## Chức năng

### Quản lý sinh viên
- Thêm sinh viên
- Xóa sinh viên
- Cập nhật thông tin sinh viên
- Tìm kiếm sinh viên

### Quản lý lớp học
- Thêm lớp học
- Xóa lớp học
- Cập nhật lớp học

### Đăng ký học phần
- Đăng ký lớp học
- Hủy đăng ký
- Xem danh sách đăng ký

## Công nghệ sử dụng

- Ngôn ngữ lập trình: C++
- Lập trình hướng đối tượng (OOP)
- Đọc/Ghi dữ liệu bằng File I/O
- Quản lý dữ liệu bằng STL Vector
- Quản lý mã nguồn bằng Git và GitHub

## Mục tiêu của dự án quản lí đăng kí học phàn cho sinh viên:

- Dự án **Hệ Thống Quản Lý Đăng Ký Học Của Sinh Viên** được xây dựng nhằm mô phỏng quy trình quản lý sinh viên, lớp học và đăng ký học trong môi trường đào tạo. Thông qua dự án, nhóm mong muốn áp dụng các kiến thức đã học về lập trình hướng đối tượng (OOP) trong ngôn ngữ C++, đồng thời rèn luyện kỹ năng thiết kế chương trình, quản lý dữ liệu và làm việc với tệp tin.

- Hệ thống cho phép người dùng thực hiện các chức năng cơ bản như quản lý thông tin sinh viên, quản lý lớp học, đăng ký học phần, hủy đăng ký học và thống kê dữ liệu. Chương trình được thiết kế theo hướng đơn giản, dễ sử dụng nhưng vẫn đảm bảo tính chính xác trong việc lưu trữ và xử lý dữ liệu.

- Bên cạnh đó, dự án còn hướng đến việc thực hành các kỹ thuật lập trình quan trọng như xây dựng lớp và đối tượng, sử dụng constructor, getter/setter, quản lý dữ liệu bằng STL Vector, xử lý ngoại lệ, đọc ghi dữ liệu bằng File I/O và tổ chức mã nguồn thành nhiều module độc lập để tăng khả năng bảo trì và mở rộng trong tương lai.

- Thông qua quá trình thực hiện dự án, các thành viên có cơ hội nâng cao kỹ năng phân tích yêu cầu, thiết kế hệ thống, lập trình nhóm và sử dụng Git/GitHub để quản lý phiên bản mã nguồn. Đây cũng là nền tảng giúp nhóm tiếp cận các hệ thống quản lý thực tế có quy mô lớn hơn trong lĩnh vực giáo dục và công nghệ thông tin.



## Yêu cầu môi trường

- Hệ điều hành: Windows / Linux
- Trình biên dịch: g++ hỗ trợ chuẩn C++11 trở lên
