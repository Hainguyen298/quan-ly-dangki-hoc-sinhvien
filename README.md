# Hệ Thống Quản Lý Đăng Ký Học Của Sinh Viên

## Mô tả
Chương trình quản lý đăng ký học phần của sinh viên được xây dựng bằng C++.

## Chức năng

### Quản lý sinh viên
- Thêm sinh viên
- Xóa sinh viên
- Cập nhật thông tin sinh viên
- Tìm kiếm sinh viên

### Quản lý lớp học
- Thêm lớp học
- Xóa lớp học
- Cập nhật lớp học

### Đăng ký học phần
- Đăng ký lớp học
- Hủy đăng ký
- Xem danh sách đăng ký

## Công nghệ sử dụng
- Ngôn ngữ: C++
- Lập trình hướng đối tượng (OOP)
- Lưu trữ dữ liệu bằng file TXT

## Cách chạy code

```bash
g++ *.cpp -o app
./app
